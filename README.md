# Space Travel

A React-powered dashboard for managing and tracking customizable spacecraft.
This frontend project displays personalized spacecraft with details and images, allows planetary travel,
and provides a clean UI for monitoring spacecraft activities.

## Features

View a list of all customized spacecraft

Detailed spacecraft cards with details and image

UI updates using React state and props

Mock loading with Mock backend data

Clean, modern layout built with React + CSS

Designed for clarity and quick scanning of planet and spacecraft lists.

Component-Based Architecture

Reusable and scalable components

Focus on clean separation of concerns and state management

## Tech Stack

React + Vite

JavaScript (ES6+)

CSS

API mocks
